# /websockets\_stats

### Parameters <a href="#parameters" id="parameters"></a>

None

### Returns <a href="#returns" id="returns"></a>

Information about the connected node

### Example

```
//Request
curl "http://<your-app-id>.ankr.com/v1a/websocket_stats" 

//Response
{
    "connections": 0,
    "subscribed_addresses": 0
}
```
